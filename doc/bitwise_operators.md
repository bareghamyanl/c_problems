#### Խնդիր 2
Գրել ծրագիր, որը մուտքում ստանում է ամբողջ տիպի թիվ, թվի վերջին 5 բիթը դարձնում է 1, ապա ստացված թիվը տպում էկրանին։
```bash
Input: 454
Output: 479
Explanation: 454 = 0000000111000110 ,after setting last 5 bits: 0000000111011111 = 479
```
